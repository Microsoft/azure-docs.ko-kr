# <a name="azure-maps-community---open-source-projects"></a>Azure Maps community 오픈 소스 프로젝트

이러한 오픈 소스 커뮤니티 기반 이니셔티브는 Azure Maps 팀에서 만들고 유지 관리 합니다. 표준 제품 또는 서비스 제공의 일부가 아닙니다.

다음 목록에는 가장 인기 있는 오픈 소스 프로젝트 및 샘플 Azure Maps 나와 있습니다.

| 프로젝트 이름 | 설명 |
|-|-|
| [Azure Maps 코드 샘플](https://github.com/Azure-Samples/AzureMapsCodeSamples) | 웹 기반 앱에서 Azure Maps를 사용 하기 위한 코드 샘플 모음입니다. |
| [Azure Maps .Gov 클라우드 코드 샘플](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Azure Government 클라우드를 통해 Azure Maps를 사용 하는 코드 샘플 모음입니다. |
| [Azure Maps & Azure Active Directory 샘플](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | Azure Maps에서 Azure Active Directory를 사용 하는 방법을 보여 주는 샘플 모음입니다. | 
| [Azure Maps를 사용 하 여 IoT 공간 분석 구현](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | 공간과 시간에서 발생하는 관련 이벤트를 추적하고 캡처하는 것은 일반적인 IoT 시나리오입니다. |
| [Azure Maps Jupyter Notebook 샘플](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | Azure Maps REST 서비스를 사용 하는 python 샘플 모음입니다. |
| [Bot 프레임 워크-관심 지점 기술](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | 관심 지점 기술은 Azure Bot Service 및 봇 프레임 워크와 함께 Azure Maps를 사용 하 여 가상 비서에 게 POI 관련 기능을 제공 합니다. |

커뮤니티에서 만든 오픈 소스 프로젝트를 포함 하는 긴 목록은 [여기](https://github.com/microsoft/Maps/blob/master/AzureMaps.md) 에서 사용할 수 있습니다.

## <a name="supportability-of-open-source-projects"></a>오픈 소스 프로젝트의 지원 가능성

다음 문은 생성 되 고 유지 관리 되는 모든 Azure Maps 오픈 소스 프로젝트 및 샘플에서 적용 됩니다.

- Azure Maps 오픈 소스 프로젝트 및 샘플은 Microsoft 및 커뮤니티에서 생성 됩니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 Microsoft와 커뮤니티에서 유지 관리 됩니다.
- 오픈 소스 프로젝트 및 샘플 Azure Maps 지원 되는 기술 및 권장 되는 기술을 사용 합니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 커뮤니티 이니셔티브로, 다른 사용자의 혜택에 대한 이니셔티브를 담당 하 고 일상적인 업무를 수행 합니다.
- Azure Maps 오픈 소스 프로젝트와 예제는 제품이 아니므로 프리미어 지원 또는 기타 공식 지원 채널에서 지원 되지 않습니다.
- Azure Maps 오픈 소스 프로젝트 및 샘플은 Microsoft에서 커뮤니티에서 지 원하는 다른 오픈 소스 프로젝트와 비슷한 방식으로 지원 됩니다.
