# <a name="azure-maps-routing-coverage"></a>Azure Maps 라우팅 적용 범위

이 문서에서는 Azure Maps 라우팅에 대한 적용 범위 정보를 제공합니다. B는 위치에서 최적 경로 검색 하면 Azure Maps에 제공할 수 있습니다 매우 정확 하 게 이동 시간 라이브 업데이트 여행 정보와 경로 지침입니다. 경로 방지 하려면 현재 트래픽, vehicle 형식 및 상태와 같은 계정 추가 검색 조건에 걸립니다. Azure Maps에 다양 한 수준의 정보 및 정확도 서로 다른 지역에 대 한 지역에 경로 최적화 하는 기능에 따라 다릅니다. 다음 표는 지역 및 정보의 종류에 요청할 수 있습니다.
 
[**지오코딩**](geocoding-coverage.md)에 대한 검사를 확인해 보세요.       
[**트래픽**](traffic-coverage.md)에 대한 검사를 확인해 보세요.  
[**렌더링**](render-coverage.md)에 대한 검사를 확인해 보세요.
