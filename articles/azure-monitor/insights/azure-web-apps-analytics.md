# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>모든 Azure Web Apps 리소스의 메트릭에 대한 분석 데이터 보기

![Web Apps 기호](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Azure Web Apps 분석 솔루션이 커뮤니티 지원으로 이동했습니다.  
>- 이 솔루션은 Azure Marketplace에 더 이상 제공되지 않지만 커뮤니티에서 지원하는 [Azure 빠른 시작 템플릿](https://azure.microsoft.com/resources/templates/webappazure-oms-monitoring/)을 통해 설치할 수 있습니다.
>- 이미 솔루션을 설치한 고객은 변경되어도 솔루션을 계속 사용할 수 있습니다.  
>- 웹 애플리케이션을 모니터링하려면 [Application Insights](../app/app-insights-overview.md)를 사용하는 것이 좋습니다.
