#### <a name="c-nuget-package"></a>C # NuGet 패키지

다음 명령을 사용 하 여 **.NET Core CLI** 에서 .NET Speech SDK를 설치할 수 있습니다 `dotnet add` .

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

다음 명령을 사용 하 여 **패키지 관리자** 에서 .NET Speech SDK를 설치할 수 있습니다 `Install-Package` .

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
