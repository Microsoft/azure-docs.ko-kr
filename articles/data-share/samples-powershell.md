# <a name="azure-powershell-samples-for-azure-data-share"></a>Azure Data Share에 대한 Azure PowerShell 샘플

다음 표에는 Azure Data Share의 Azure PowerShell 샘플 스크립트에 대한 링크가 나와 있습니다.

|PowerShell 샘플|Description|
|---|---|
|[새 데이터 공유 계정 만들기](scripts/powershell/create-new-share-account-powershell.md)| 이 PowerShell 스크립트는 새 데이터 공유 계정을 만듭니다. |
|[새 데이터 공유 만들기](scripts/powershell/create-new-share-powershell.md)| 이 PowerShell 스크립트는 새 데이터 공유를 만듭니다. |
|[공유하기 위해 데이터 세트 추가](scripts/powershell/add-datasets-powershell.md)| 이 PowerShell 스크립트는 기존 데이터 공유 계정에 Blob 데이터 세트를 추가합니다. |
|[공유 동기화 설정 및 보기](scripts/powershell/set-view-synchronizations-powershell.md)| 이 PowerShell 스크립트는 기존 데이터 공유의 동기화를 만들고 봅니다. |
|[데이터 공유 초대 만들기](scripts/powershell/create-share-invitation-powershell.md)| 이 샘플 PowerShell 스크립트는 기존 데이터 공유에 대한 액세스 초대를 만듭니다. |
|[보낸 공유 초대 보기](scripts/powershell/view-sent-invitations-powershell.md)| 이 샘플 PowerShell 스크립트는 기존 데이터 공유에서 보낸 초대 목록을 검색합니다. |
|[수신된 공유 초대 수락](scripts/powershell/accept-share-invitations-powershell.md)| 이 샘플 PowerShell 스크립트는 데이터 공유 초대를 수락합니다.|
|[데이터 공유에 대한 세부 정보 보기](scripts/powershell/view-share-details-powershell.md)| 이 샘플 PowerShell 스크립트는 데이터 공유에 대한 세부 정보를 나열하고 검색합니다. |
|[공유 데이터 사용 모니터링](scripts/powershell/monitor-usage-powershell.md)| 이 샘플 PowerShell 스크립트는 전송된 공유 데이터의 사용을 모니터링합니다. |
|[스냅샷 트리거 만들기 및 보기](scripts/powershell/create-view-trigger-powershell.md)| 이 샘플 PowerShell 스크립트는 공유의 스냅샷 트리거를 만듭니다.
