## <a name="data-box-heavy-limits"></a>Data Box Heavy 제한

- 노드당 최대 10억 파일을 저장할 수 Data Box Heavy.
- Data Box Heavy는 클라우드의 노드당 최대 512의 컨테이너 또는 공유를 지원 합니다. 사용자 공유 내의 최상위 디렉터리는 클라우드의 컨테이너 또는 Azure 파일 공유로 사용 됩니다.
