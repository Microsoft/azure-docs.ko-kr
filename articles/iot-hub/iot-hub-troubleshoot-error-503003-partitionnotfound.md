# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

이 문서에서는 **503003 PartitionNotFound** 오류의 원인과 해결 방법을 설명합니다.

## <a name="symptoms"></a>증상

IoT Hub에 대한 요청이 실패하고 **503003 PartitionNotFound** 오류가 표시됩니다.

## <a name="cause"></a>원인

이 오류는 IoT Hub 내부 오류이며 일시적일 수 있습니다. [IoT Hub 내부 서버 오류 원인](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause)을 참조하세요.

## <a name="solution"></a>솔루션

[IoT Hub 내부 서버 오류에 대한 해결 방법](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution)을 참조하세요.
