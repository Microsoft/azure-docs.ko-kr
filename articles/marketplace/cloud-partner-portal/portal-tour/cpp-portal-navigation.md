# <a name="cloud-partner-portal-navigation"></a>Cloud 파트너 포털 탐색

[Cloud 파트너 포털](https://cloudpartner.azure.com)에서 탐색하는 두 가지 기본 UI(사용자 인터페이스) 메커니즘은 상단 메뉴 모음과 왼쪽 탐색 창입니다.  이러한 UI 요소는 고정적이며 탐색하는 포털의 어떤 페이지에서도 사용할 수 있습니다.


## <a name="menu-bar"></a>메뉴 모음

메뉴 모음은 포털 사이트의 위쪽에서 실행됩니다.  여기에는 다음과 같은 4개의 활성 영역에 있습니다.

![메뉴 모음](./media/top-menubar1.png)

이러한 영역을 클릭하면 다음 작업이 수행됩니다.

1. 제목 영역 - 포털 홈 페이지로 이동합니다.
2. **도움말** 단추 - 사용자를 지원하는 리소스를 제공하는 도움말 메뉴가 표시됩니다.
3. **피드백** 단추 - **피드백 보내기** 양식이 표시됩니다.
4. **게시자** 단추 - 사용자가 자신의 프로필을 관리할 수 있는 메뉴가 표시됩니다.

이러한 메뉴 및 형식은 다음에 설명되어 있습니다.

### <a name="help-menu"></a>도움말 메뉴

**도움말** 메뉴는 Cloud 파트너 포털을 사용하여 사용자를 지원하는 리소스를 제공합니다.

![상단 메뉴 모음](./media/top-menubar2.png)

도움말 메뉴에는 다음 항목이 포함됩니다.

- **설명서** - 브라우저에서 새 탭이 시작되고 Cloud 파트너 포털에 대한 설명서 홈 페이지로 이동합니다. 
- **포럼** - 새 탭이 시작되고 AppSource 및 Azure Marketplace용 파트너 커뮤니티에 대한 설명서 홈 페이지로 이동합니다.
- **지원** - 새 탭이 시작되고 인시던트 보고서를 만들 수 있는 Marketplace 게시를 위한 지원 페이지로 이동합니다.  Microsoft는 모든 피드백을 심사 및 검토하며, 언제나 고객의 개인 정보를 보호합니다. 고객이 자세한 정보를 요청하거나 질문하는 경우에만 연락을 드립니다.


### <a name="send-us-feedback-form"></a>피드백 보내기 양식

**피드백 보내기** 양식은 Marketplace 게시자 온보딩 팀에 피드백을 제공하는 직접적인 방법을 제공합니다.  연락을 드립니다.**내 피드백과 관련하여 연락 허용** 확인란을 선택한 경우에만 추가 정보에 관해 연락을 드립니다.

![사용자 의견 양식](./media/feedback-form.png)

> [!TIP]
> Cloud 파트너 포털에서 특정 페이지에 대한 피드백이 있으면 **피드백** 단추를 클릭하기 전에 해당 페이지로 이동하세요.  세션의 UI 상태를 캡처할 수 있도록 피드백을 제출하기 전에 **스크린샷 포함** 확인란이 선택되어 있는지 확인하세요. 


### <a name="publisher-menu"></a>게시자 메뉴

이 메뉴를 사용하면 게시자 프로필 페이지에 직접 액세스하거나, Microsoft 게시자 계약을 보거나 현재 세션에서 로그아웃하는 등 프로필 및 세션을 관리할 수 있습니다. 

![게시자 메뉴](./media/publisher-menu.png)
