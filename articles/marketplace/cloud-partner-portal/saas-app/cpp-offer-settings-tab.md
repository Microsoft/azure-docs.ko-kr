# <a name="saas-application-offer-settings-tab"></a>SaaS 애플리케이션 제품 설정 탭

이 제품의 제품 설정을 구성하는 방법을 설명합니다.

**SaaS 앱 > 새 제품** 페이지는 **제품 설정** 탭에 초점을 맞춰 열립니다. 

다음 화면 캡처에 표시된 것처럼 제품 설정 탭을 사용하여 **제품 ID**를 구성합니다. 필드 이름에 추가된 별표(*)는 필수를 나타냅니다.

![제품 설정 탭](./media/saas-new-offer.png)

## <a name="offer-identity-settings"></a>제품 ID 설정

제품 ID 아래에서 다음 표에서 설명하는 필드에 대한 정보를 제공해야 합니다. 


|  **필드 이름**   |  **설명**  |
|  ---------------   |  ---------------  |
|    제품 ID  |  게시자 프로필에 있는 제품의 고유 식별자입니다. 이 ID는 제품 URL 및 청구 보고서에서 볼 수 있습니다. 소문자 영숫자 문자 또는 대시(-)로만 구성할 수 있습니다. ID는 대시로 끝내면 안 되며 최대 50문자로 제한됩니다. 참고로 이 필드는 제품이 라이브 상태가 되면 잠깁니다. 예를 들어, Contoso 게시자가 제품 ID가 sample-vm인 제품을 게시하면 Azure Marketplace에 다음과 같이 표시됩니다. https://azuremarketplace.microsoft.com/marketplace/apps/contoso.sample-vm?tab=Overview                 |
|  게시자 ID    |  게시자 ID는 Marketplace에서 고유 식별자입니다. 모든 제품은 게시자 ID에 연결되어야 합니다. 제품이 저장되면 게시자 ID를 수정할 수 없습니다.                |
|  이름      |   제품에 대한 표시 이름입니다. Azure Marketplace 및 Azure Portal에 표시되는 이름입니다. 최대 50문자를 포함할 수 있습니다. 제품의 인식 가능한 브랜드 이름을 포함합니다. 회사 이름을 포함하는 것이 마케팅 전략인 경우를 제외하고 여기에 회사 이름을 포함하지 마세요. 이 제품을 자신의 고유 웹 사이트에 마케팅하는 경우 이 이름은 웹 사이트에 표시되는 이름과 같아야 합니다.               |

**저장**을 클릭하여 진행 상황을 저장합니다.
