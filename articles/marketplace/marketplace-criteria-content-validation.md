# <a name="azure-marketplace-listing-guidelines"></a>Azure Marketplace 목록 가이드라인

이 문서에는 Azure Marketplace 목록에 추가된 새 제품 및 서비스의 검토자를 위한 검사 목록 및 요구사항 가이드라인이 포함되어 있습니다.

모든 제품은 아래에 표시된 [모든 목록 요구 사항](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#all-listing-requirements)을 충족해야 합니다. 첨부된 [평가판](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#trial-offer-requirements)이 있는 제품, [SaaS 앱](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#saas-apps-requirements), [컨테이너](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#container-offer-requirements) 및 [컨설팅 제품](https://docs.microsoft.com/azure/marketplace/marketplace-criteria-content-validation#consulting-offer-requirements)을 포함하여 특정 목록 유형에 대해서는 추가 요구 사항 및 검사 목록이 제공됩니다.


## <a name="all-listing-requirements"></a>모든 목록 요구 사항

|  | 목록 요소 | 기본 요구 사항 | 최적 요구 사항 |
|:--- |:--- |:--- |:--- |
| 1 | 잠재 대상 고객 | 잠재 대상 고객이 구성되어 있음| OCP 카탈로그에 파트너 솔루션 탭에도 나열된 잠재 대상 고객 CRM 정보가 있음 | 
| 2 | 제품 이름 | 솔루션 제공을 설명함 파트너 웹 사이트의 솔루션 온라인 홍보와 일치함 | 주요 검색어를 포함함 |
| 3 | 로고 | 로고가 올바르게 표시됨 | 로고가 올바르게 표시되고 Azure Portal의 큰 형식 이미지에 해당하는 [대표 이미지](https://docs.microsoft.com/azure/marketplace/cloud-partner-portal-orig/cloud-partner-portal-solution-template-offer-publish)를 포함함 |
| 4 |  제품 설명 | <br>• 2-3개 단락 <br>• 솔루션 제공을 한눈에 쉽게 이해할 수 있음 <br>• 제품 설명에 맞춤법 및 문법 오류가 없음 <br>• 제품 설명은 종합적이고 대상 그룹, 사용자 유형, 유용한 이유(가치 명제)를 포함함 <br>• 제품 설명은 이해하기 쉬운 짧은 문장을 사용하여 단락 형식으로 제공됨 | <br>• 대상 산업이 간략하게 설명됨(해당하는 경우) <br>• 각 단락 머리글에서 한 문장이나 구로 이후에 나오는 콘텐츠를 요약하고 글머리 기호(해당하는 경우)를 사용하여 주요 혜택을 강조하는 좋은 스타일 서식을 지정함. 여기서 목표는 독자가 단락을 읽지 않고도 보기 편리한 형식으로 제품을 한눈에 이해할 수 있게 하는 것입니다. <br>• 각 단락 사이에 간격이 있으며 자동차 브로슈어처럼 읽힘. 즉, 기술적인 기능이나 코드 용어가 아니라 간단한 용어로 제공을 설명하고 종합적입니다. | 
| 5 | 범주 및 산업 | <br>• 범주가 솔루션 제공 기능과 일치함 <br>• 솔루션에 맞지 않는 범주로 확장하지 않음 <br>• 최적 산업 또는 선택한 모든 산업(최적 산업이 아닌 경우에도) | <br>• 최대 3개 범주 <br>• 최대 3개 산업 |
| 6 | 이미지 | <br>• 이미지가 필수는 아니지만, 제공된 경우 표시되고 고해상도 이미지여야 함 <br>• 이미지 요구 사항은 CPP 및 여기에 나열됨 <br>• 스크린샷에 포함된 텍스트는 선명한 이미지로 읽기 쉬움 | 솔루션 제공을 한눈에 쉽게 이해할 수 있음 |
| 7 | 동영상 | <br>• 동영상이 필수는 아니지만, 제공된 경우 오류 없이 재생되어야 함 <br>• 제공된 경우 마이그레이션 솔루션을 시연하지 않는 한 경쟁업체를 가리킬 수 없음 |<br>• 3분 이상이 이상적임 <br>• 동영상 콘텐츠를 통해 솔루션 제공을 쉽게 이해할 수 있음 <br>• 솔루션 기능 데모 |
| 8 | 목록 상태(판촉물 문구) | <br>다음 유형 중 하나로 레이블이 지정되어야 함 <ul>o 연락처</ul><ul>o 평가판/지금 평가판 다운로드/평가판 시작/시험 사용</ul><ul>o 지금 구입/지금 다운로드</ul> | 고객이 사용 가능한 다음 단계를 쉽게 이해할 수 있음 <ul>1. 평가판 사용</ul> <ul>2. 지금 구입</ul><ul>3. 메일 또는 전화 번호를 통해 연락하여 POC, 평가 또는 브리핑 준비</ul> | 
| 9 | 솔루션 가격 | 솔루션 가격 탭/세부 정보가 있고 파트너 솔루션 제공의 현지 통화로 지정되어야 함 | 고객이 선택할 수 있도록 계층 가격 책정으로 여러 청구 옵션이 제공됨 |    
| 10 | 자세한 정보 | 맨 아래 링크(설명 아래에 있고 왼쪽의 마켓플레이스 링크 아님)를 클릭하면 솔루션에 대한 자세한 정보로 이동하며, 공개적으로 사용 가능하고 올바르게 표시됨 |   파트너 홈페이지 사이트가 아닌 특정 항목(예: 파트너 사이트의 사양 페이지)에 대한 링크임 | 
| 11 | 솔루션 지원 및 도움말 | 다음 중 하나 이상에 대한 링크: <br>o 전화 번호 <br>o 메일 지원 <br>o 채팅 에이전트 <br>o 커뮤니티 포럼 |<br>• 모든 지원 방법이 나열됨: 전화 및 메일 지원 <br>• 평가판 또는 시험 사용 기간 동안 유료 지원이 무료로 제공됨 | 
| 12 | 법적 정보 | 공용 URL을 통해 정책 또는 사용 약관이 제공됨 |  |
