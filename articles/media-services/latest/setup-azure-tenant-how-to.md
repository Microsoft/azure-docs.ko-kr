# <a name="how-to-find-your-azure-tenant-id"></a>Azure 테넌트 ID를 찾는 방법

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

[!INCLUDE [quickstarts-free-trial-note](../../../includes/quickstarts-free-trial-note.md)]

## <a name="find-your-tenant-id-in-the-portal"></a>포털에서 테넌트 ID 찾기

1. [Azure Portal](https://portal.azure.com)에 로그인합니다.
1. Azure 서비스에서 Azure Active Directory 아이콘을 선택합니다. Active Directory 페이지가 열립니다.
1. 테넌트 ID는 **기본 정보** 아래에 있습니다.
1. 테넌트 ID를 복사하여 나중에 사용할 수 있도록 선택한 텍스트 문서에 붙여넣습니다.
