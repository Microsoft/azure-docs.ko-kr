# <a name="azure-network-security"></a>Azure 네트워크 보안
## <a name="abstract"></a>요약
Azure 네트워크 서비스는 기본적으로 유연성, 가용성, 탄력성, 보안 및 무결성을 최대화합니다. 이 백서는 Azure의 네트워킹 기능에 대한 세부 정보를 제공합니다. 또한 고객이 Azure의 기본 보안 기능을 사용하여 정보 자산을 보호하는 방법을 설명합니다.
이 백서의 대상 사용자는 다음과 같습니다.
- Azure에서 사용 가능하고 지원되는 보안 솔루션을 찾고 있는 기술 관리자, 네트워크 관리자 및 개발자
- Azure 공용 클라우드의 네트워크 보안과 관련된 Azure 기술 및 서비스에 대한 요약 정보를 원하는 SME 또는 비즈니스 프로세스 실무 담당자
