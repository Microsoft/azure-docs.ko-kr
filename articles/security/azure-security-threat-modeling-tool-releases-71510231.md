# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Threat Modeling Tool 업데이트 릴리스 7.1.51023.1 - 2018/11/1

[GA 릴리스 정보](./azure-security-threat-modeling-tool-releases-71509112.md)에 원래 언급된 대로, Preview 버전(7.1.50911.2 이전 버전의 미리 보기 클라이언트)의 사용자가 지원되는 GA 릴리스로 업그레이드하도록 요구하는 Microsoft Threat Modeling Tool 업데이트(7.1.51023.1)가 릴리스되었습니다. 이 릴리스에는 새로운 기능이나 수정 사항이 포함되어 있지 않습니다.

- Preview 버전의 사용자가 클라이언트를 열면 자동으로 업그레이드가 다운로드됩니다. 새 업데이트를 설치하지 않도록 선택하면 도구의 Preview 버전이 닫힙니다.
- 도구의 GA 버전 사용자에게는 업그레이드 여부를 선택하라는 메시지가 표시됩니다.
- 도구의 2016 버전 사용자는 영향을 받지 않습니다.

## <a name="feature-changes"></a>기능 변경

## <a name="system-requirements"></a>시스템 요구 사항

- 지원되는 운영 체제
  - Microsoft Windows 10
- 필요한 .NET 버전
  - .NET 3.5.2
- 추가 요구 사항
  - 도구와 템플릿에 대한 업데이트를 받으려면 인터넷 연결이 필요합니다.
