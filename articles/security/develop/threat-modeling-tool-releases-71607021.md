# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Threat Modeling Tool 업데이트 릴리스 7.1.60702.1 - 7/2/2019

Microsoft TMT(Threat Modeling Tool) 버전 7.1.60702.1은 2019년 7월 2일 출시되었으며 이 버전에 포함된 변경 사항은 다음과 같습니다.

- 내게 필요한 옵션 기능 향
- 버그 수정
- 오픈 소스 커뮤니티 제공 스텐실 세트 포함

## <a name="feature-changes"></a>기능 변경

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>오픈 소스 커뮤니티에서 제공하는 새 의료 디바이스 스텐실 세트 사용 가능

의료 디바이스 모델링을 위한 스텐실 세트가 오픈 소스 커뮤니티에서 제공되었습니다. 업데이트하면 새 스텐실 세트가 템플릿 선택 드롭다운 메뉴에 표시됩니다. 템플릿에 스텐실이나 콘텐츠를 기여하는 방법에 대한 자세한 내용은 프로젝트의 [GitHub 페이지](https://github.com/Microsoft/threat-modeling-templates)에 있는 정보를 검토하세요.

![모델 유효성 검사 옵션](./media/threat-modeling-tool-releases-71607021/tmt-template-selection.png)

## <a name="system-requirements"></a>시스템 요구 사항

- 지원되는 운영 체제
  - [Microsoft Windows 10 1주년 업데이트](https://blogs.windows.com/windowsexperience/2016/08/02/how-to-get-the-windows-10-anniversary-update/#HTkoK5Zdv0g2F2Zq.97) 이상
- 필요한 .NET 버전
  - [.Net 4.7.1](https://go.microsoft.com/fwlink/?LinkId=863262) 이상
- 추가 요구 사항
  - 도구와 템플릿에 대한 업데이트를 받으려면 인터넷 연결이 필요합니다.
