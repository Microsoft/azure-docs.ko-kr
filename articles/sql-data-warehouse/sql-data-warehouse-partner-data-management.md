# <a name="sql-data-warehouse-data-management-partners"></a>SQL Data Warehouse 데이터 관리 파트너
데이터 웨어하우스 솔루션을 만들려면 다양한 업계 최고의 도구에서 선택합니다. 이 문서에서는 Azure SQL Data Warehouse를 지원하는 데이터 관리 도구 및 솔루션을 제공하는 Microsoft 파트너 기업을 중점적으로 소개합니다.

## <a name="data-management-partners"></a>데이터 관리 파트너
| 파트너 | 설명 | 웹 사이트/제품 링크 |
| ------- | ----------- | -------------------- |
| ![Coffing Data Warehousing][1] |**Coffing Data Warehousing**<br>Coffing Data Warehousing은 10년 동안 쿼리 시스템을 전용으로 디자인한 도구인 Nexus Chameleon을 제공합니다. Nexus는 SQL Data Warehouse에 대한 쿼리 도구로 사용 가능합니다. Nexus를 사용하여 사내 및 클라우드 컴퓨터에서 쿼리하여 다른 플랫폼에서 데이터를 조인합니다. 마우스로 클릭하여 보고합니다! |[제품 페이지][coffing_website]<br> |
| ![Inbrein][2] |**Inbrein MicroERD**<br>Inbrein MicroERD는 정확한 데이터 모델을 만들고, 데이터 중복을 최소화하고, 생산성을 향상시키고, 표준을 준수하는 데 필요한 도구를 제공합니다. 광범위한 사용자 환경을 기반으로 개발된 UI를 사용하여, 모델러는 DB 모델에서 쉽고 편리하게 작업할 수 있습니다. 프롬프트 기능 향상 및 업데이트를 통해 지속적으로 MicroERD의 새로운 기능과 향상된 기능을 즐길 수 있습니다. |[제품 페이지][inbrein_website]<br> |
| ![Infolibrarian][3] |**Infolibrarian (메타 데이터 관리 서버)**<br>InfoLibrarian 카탈로그는 메타데이터를 저장하고 관리하여 데이터 관리의 주요 문제점을 해결하는 데 도움이 됩니다. 또한 Infolibrarian은 다양한 도구 및 기술 집합에서 메타데이터를 관리하고 게시하기 위한 솔루션으로 메타데이터 관리, 데이터 관리 및 자산 관리 소프트웨어 솔루션을 제공합니다. |[제품 페이지][infolibrarian_website]<br> [Azure Marketplace][infolibrarian_marketplace]<br> |
| ![Redpoint Global][4] |**RedPoint Data Management**<br>RedPoint Data Management 마케터를 자신의 모든 데이터 크로스 채널 고객 계약 추진 하면서 구조적 및 비구조적된 데이터 관리를 적용할 수 있습니다. Azure SQL Data Warehouse를 활용하여, RedPoint 사용자는 오늘날의 모든 채널 사용 고객과 의견을 교환하는 데 필요한 하이퍼-개인 설정 및 컨텍스트 상호작용을 제공하도록 모든 구조적 및 비구조적 데이터의 값을 최대화할 수 있습니다. 끌어서 놓기 인터페이스를 사용하면 데이터 관리 프로세스를 쉽게 설계하여 실행할 수 있습니다. |[제품 페이지][redpoint_dm_website]<br> [Azure Marketplace][redpoint_dm_marketplace]<br> |
| ![Sentry One][5] |**SentryOne (DW Sentry)**<br>지능형 데이터 이동 대시보드 및 이벤트 달력을 사용하면 워크로드에 영향을 주는 작업을 항상 정확히 알 수 있습니다. 데이터를 로드, 백업 또는 복원하기 위해 실행 중인 쿼리 및 작업에 대한 가시성을 제공하도록 설계되었기 때문에 Azure 리소스의 대부분을 사용하는 것에 대해 걱정할 필요가 없습니다. |[제품 페이지][sql_sentry_website]<br>[Azure Marketplace][sql_sentry_marketplace]<br> |
