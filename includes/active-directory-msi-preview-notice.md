Azure 리소스에 대한 관리 ID는 Azure Active Directory의 기능입니다. [Azure 리소스에 대한 관리 ID를 지원하는 각 Azure 서비스](/azure/active-directory/managed-identities-azure-resources/services-support-msi)는 자체 타임라인을 따릅니다. 시작하기 전에 리소스의 관리 ID [가용성](/azure/active-directory/managed-identities-azure-resources/services-support-msi) 상태와 [알려진 문제](/azure/active-directory/managed-identities-azure-resources/known-issues)를 검토하세요.
