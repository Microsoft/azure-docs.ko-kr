보안 제어는 서비스에서 보안 취약성을 예방, 감지 및 대응 하는 기능을 제공 하는 Azure 서비스의 품질 또는 기능입니다.

각 컨트롤에 대해 "예" 또는 "아니요"를 사용 하 여 서비스에 대해 현재 사용 중인지 여부를 나타냅니다. "N/A"는 서비스에 적용할 수 없는 컨트롤입니다. 또한 특성에 대한 추가 정보를 제공 하는 링크를 제공할 수도 있습니다.
