VM(가상 머신)에는 VM이 배포된 서브넷에 따라 사용자가 지정한 범위에서 개인 IP 주소가 자동으로 할당됩니다. VM은 VM이 삭제될 때까지 주소를 유지합니다. Azure는 VM을 만드는 서브넷에서 사용 가능한 다음 개인 IP 주소를 동적으로 할당합니다. VM에 할당된 서브넷의 특정 IP 주소를 원하는 경우 고정 IP 주소를 할당합니다.
