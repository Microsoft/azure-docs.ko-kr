상승된 권한으로 PowerShell 콘솔을 엽니다.

Azure PowerShell을 로컬로 실행하는 경우 Azure 계정에 연결합니다. *Connect-AzAccount* cmdlet에서 자격 증명을 입력하라는 메시지를 표시합니다. 인증한 다음, Azure PowerShell에 사용할 수 있도록 계정 설정을 다운로드합니다. 대신 Azure Cloud Shell를 사용 하는 경우 *AzAccount*를 실행할 필요가 없습니다. Azure Cloud Shell는 Azure 계정에 자동으로 연결 됩니다.

```azurepowershell
Connect-AzAccount
```

둘 이상의 구독이 있는 경우 Azure 구독 목록을 가져옵니다.

```azurepowershell-interactive
Get-AzSubscription
```

사용할 구독을 지정합니다.

```azurepowershell-interactive
Select-AzSubscription -SubscriptionName "Name of subscription"
```
