Azure PowerShell cmdlet을 컴퓨터에 로컬로 설치 하 고 실행할 수도 있습니다. PowerShell cmdlet은 자주 업데이트 됩니다. 최신 버전을 실행 하지 않는 경우 지침에 지정 된 값이 실패할 수 있습니다. 컴퓨터에 설치 된 Azure PowerShell 버전을 찾으려면 `Get-Module -ListAvailable Az` cmdlet을 사용 합니다. 설치 또는 업데이트 하려면 [Azure PowerShell 모듈 설치](/powershell/azure/install-az-ps)를 참조 하세요.
